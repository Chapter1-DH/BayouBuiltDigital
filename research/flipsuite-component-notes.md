## Overview

FlipSuite Studios is a React-based suite of reseller/flipping tools. The main app uses JSX components with a modern UI featuring:

- Dark theme with accent colors per app category
- Card-based layout for app showcase
- Responsive grid design
- Perpetual fallback licensing model prominently featured

## Key Differentiators

1. **Perpetual Fallback Licensing** - Subscribe for 12 months, keep a perpetual license to that version
2. **Multi-Platform** - Targets eBay, Mercari, Poshmark, FB Marketplace (not just Amazon FBA)
3. **Offline-First** - Works at thrift stores and flea markets with poor connectivity
4. **Unified Suite** - Shared inventory, analytics, and listing engine across all apps
